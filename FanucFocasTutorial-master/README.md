# 파낙 포커스 자동 옵셋 시스템

## 프로젝트 개요
이 프로젝트는 파낙(FANUC) CNC 기계의 공구 옵셋을 자동으로 관리하고 보정하는 시스템입니다. 측정 데이터를 기반으로 공구 보정값을 자동으로 계산하고 적용하여 가공 정밀도를 향상시킵니다.

## 주요 기능

### 1. 자동 옵셋 기능
- 측정 데이터 자동 로드 및 분석
- 공차 범위 기반 판정 (OK, HIGH, LOW)
- 연속 측정값 일관성 검증
- 자동 보정량 계산 및 적용
- 외경 상하 차이 자동 보정

### 2. 장비 모니터링
- 프로그램 정보 표시
- 공구/옵셋 정보 모니터링
- 스핀들 정보 (회전수, 부하) 표시
- 축 정보 (X, Z 위치) 표시
- 이송 정보 및 가공/운전 시간 모니터링

### 3. 데이터 관리
- SQLite 데이터베이스 연동
- 측정 데이터 자동 갱신
- 측정 이력 관리
- 보정 이력 기록

### 4. 사용자 인터페이스
- 직관적인 그리드 기반 데이터 표시
- 자동/수동 옵셋 모드 지원
- 경고 메시지 자동 표시 및 타이머 기능
- IP 관리 및 연결 상태 표시

## 기술 스택
- C# .NET Framework
- Windows Forms
- SQLite 데이터베이스
- FANUC FOCAS 라이브러리

## 업데이트 필요 사항

### 1. 백그라운드 기능 추가
- 시작 버튼의 여부에 따라 백그라운드에서 자동으로 실행되는 기능 구현
- 백그라운드 모드에서 자동 보정 처리 및 로그 기록
- 시스템 트레이 아이콘 및 알림 기능 추가

### 2. 실제 공구 옵셋 데이터 보정
- FANUC FOCAS 라이브러리를 통한 실제 CNC 기계 공구 옵셋 데이터 직접 수정
- 보정 적용 후 검증 및 피드백 메커니즘 구현
- 보정 이력 데이터베이스 저장 및 관리

### 3. 측정 데이터 갱신 시 처리 프로세스 점검
- 데이터 갱신 시 처리 흐름 최적화
- 데이터 일관성 검증 강화
- 오류 처리 및 복구 메커니즘 개선
- 측정 데이터 변화 추적 및 분석 기능 추가
